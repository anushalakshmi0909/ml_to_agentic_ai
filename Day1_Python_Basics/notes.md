# Day 1 — Python Basics for Machine Learning

## Objective
Build a strong foundation in Python programming to support future work in Machine Learning, LLMs, and Agentic AI systems.

## Topics Covered
- Variables
- Data Types
- Lists
- Dictionaries
- Loops

## Key Learnings
Python is the primary programming language used in the machine learning ecosystem. Understanding basic data structures and control flow is essential before working with ML libraries such as NumPy, Pandas, and Scikit-learn.

Lists allow storing and iterating through collections of data, while dictionaries are commonly used to represent structured information like JSON responses and configuration settings.
